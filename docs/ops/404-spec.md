# 404 Page Spec

## Behavior
- Display a friendly 404 headline and short explanation.
- Offer a primary CTA to book (`https://booking.doulab.net/`).
- Provide secondary links: Services, Docs, Contact, Home.
- No inline styles; use shared components and existing card styles.

## Content
- Headline: playful and concise (e.g., "This page escaped the lab.").
- Body: clarify the page does not exist and suggest next steps.

## Link set
- Primary CTA: Book (booking.doulab.net)
- Secondary: `/services`, `/docs/research-resources`, `/contact`, `/`
