# GDPR / Swiss nFADP Baseline (Doulab)

## Controller
- Doulab GmbH (Switzerland) acts as the data controller for doulab.net.
- Contact: legal@doulab.net

## Data categories processed
- Website browsing data (IP, device/browser, language, referral URL) via privacy-respecting analytics.
- Contact inquiries (name, email, organization, message content).
- Booking details (name, email, requested time) handled through booking links that redirect to Google Calendar appointment scheduling.
- Project/engagement data shared during diagnostics, workshops, and advisory sessions.

## Purposes and lawful basis (GDPR-aligned)
- Service delivery and pre-contract steps (contract performance / pre-contract measures).
- Operational communications and support (legitimate interest).
- Security, availability, and fraud prevention (legitimate interest).
- Legal obligations (records and compliance).

## Processors / subprocessors
- Cloudflare Pages (hosting, CDN, web analytics).
- GitHub (source code and CI/CD).
- Google Calendar appointment links (external scheduling for bookings).
- Stripe Checkout for payments (if applicable to fixed-fee services).

## Tracking stance
- Default stance: no non-essential tracking or marketing cookies unless explicitly configured.
- If analytics is present, it is privacy-respecting and disclosed publicly.

## Retention principles
- Retain only what is needed for service delivery, compliance, and support.
- Engagement records are kept for a limited period, then deleted or anonymized.
- Requests to delete or export data are honored when legally permissible.

## User rights
- Access, rectification, deletion, and restriction where applicable.
- Objection to certain processing and data portability where required.
- Requests via legal@doulab.net with identity verification as needed.

## Cross-border processing
- Cloudflare and Google may process data outside Switzerland/EU.
- Transfers rely on standard safeguards (e.g., SCCs) when required.

## Compliance note
- This baseline aligns with GDPR principles and Swiss nFADP requirements.
- This document is operational guidance and is not legal advice.
